# 実運用手順（配置先: C:\Users\Owner\Desktop\script\画像分割カットツール）

このフォルダ配下に、実運用ツール一式を作成済みです。  
**あなたの指定パスそのまま**で使えます。

## できたもの

- `desktop_tool/` : 本体（画像分割 + DeepL/Gemini翻訳GUI）
- `起動.bat` : ルートからワンクリック起動

## 使い方

1. `C:\Users\Owner\Desktop\script\画像分割カットツール\起動.bat` をダブルクリック
2. 初回は自動で仮想環境作成・依存インストール
3. ツール起動後、画像分割または翻訳を実行

## 備考

- APIキーは `desktop_tool/settings.json` に保存されます。
- 画像出力先はGUIで自由に変更できます。


## よくある質問

### Q. これは自分でダウンロードして入れたらいいの？
A. **このフォルダを丸ごと置くだけでOK**です。追加で手作業が必要なのは、最初のPython導入だけです。  
その後は `かんたん開始.bat` をダブルクリックすれば、必要なライブラリは自動で入ります。
